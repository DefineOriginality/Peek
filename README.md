# Peek
A Chrome extension for TETR.IO that quickly returns information about an opponent.

As of v1.1.0, this extension is not on the Chrome Web Store. I do have plans to do so if it makes installation easier, but for now, you will have to follow the below instructions to install:

1. Go to Releases and download the latest version, then extract it.
2. Visit chrome://extensions in your browser.
3. Click the "Developer Mode" toggle in the top right corner, which should reveal a "Load unpacked" button, now visible in the top left corner. Click it and select the folder named "Peek (SELECT THIS FOLDER)", which is INSIDE the folder you just extracted (NOT the folder that has version name on it). If it asks you for any permissions, accept them.
4. (Optional) You're done at this point, but I recommend pinning the extension so you can use it with one click at all times. That's it!

I wrote this project entirely by hand (discounting Google's built-in AI) with like 3 hours of former Javascript knowledge, so please excuse me if there's any hiccups.
